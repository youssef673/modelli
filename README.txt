# Sito GitHub Pages - Versione migliorata

## Cosa è stato migliorato
- grafica più moderna
- barra di avanzamento durante lo scroll
- menu mobile
- sezione quiz finale
- blocchi "come dirlo a voce"
- confronto più chiaro
- layout migliore per una presentazione scolastica

## File inclusi
- `index.html`
- `style.css`
- `script.js`

## Come pubblicarlo
1. Crea un repository su GitHub
2. Carica i file
3. Vai su `Settings > Pages`
4. Seleziona `Deploy from a branch`
5. Scegli `main` e `/root`
6. Salva
