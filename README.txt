# Sito GitHub Pages - Modello di McCall e Modello di Boehm

## File inclusi
- `index.html`
- `style.css`
- `script.js`

## Come pubblicarlo su GitHub senza installare nulla
1. Crea un repository su GitHub.
2. Carica questi 3 file nel repository.
3. Vai su **Settings > Pages**.
4. In **Build and deployment**, scegli:
   - **Source**: Deploy from a branch
   - **Branch**: `main` e cartella `/root`
5. Salva.
6. Dopo pochi minuti il sito sarà online.

## Personalizzazione veloce
- Cambia il titolo dentro `index.html`
- Cambia i colori dentro `style.css`
- Modifica i testi della presentazione dentro `index.html`
